# ME Harness Boundary

Codex is the runtime harness.

ME is the semantic harness.

## Technical Harness

A technical harness runs the model loop, tools, context, permissions,
sessions, and host integration. Codex App is ME's intended v0.x
technical harness.

## Semantic Harness

ME controls what agent output is allowed to make count.

ME does not run the model. ME defines the semantic state machine and
accepts only legal transactions into canonical state.

Use this public wording:

```text
Codex is the runtime harness.
ME is the semantic harness.
```

Avoid saying:

```text
ME is an agent harness.
```

That phrase conflicts with the conventional technical meaning.
